# FUTURE_DS_01 – Data Science & Analytics Internship

## Task 1: Business Sales Dashboard from E-Commerce Data

### Objective
Analyze e-commerce transaction data and build an interactive sales dashboard to identify revenue trends, high-performing product categories, and transaction behavior to support business decision-making.

### Dataset
E-Commerce Transactions Dataset sourced from Kaggle, containing transaction-level data such as product categories, payment methods, transaction dates, and purchase amounts.

### Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel (for initial review)

### Key Analysis Performed
- Data cleaning and preprocessing (duplicates, null values, invalid amounts)
- KPI creation: Total Sales, Total Transactions, Average Transaction Value, Last Month Sales
- Revenue analysis by product category
- Transaction analysis by payment method
- Monthly sales trend analysis

### Key Insights
- Electronics and Clothing are the highest revenue-generating categories
- Monthly sales fluctuate between approximately $2.0M–$2.2M, indicating stable demand without strong seasonality
- Digital payment methods dominate transactions, showing low reliance on cash payments

### Deliverable
An interactive Power BI dashboard with filters, KPIs, trend analysis, and business insights.
